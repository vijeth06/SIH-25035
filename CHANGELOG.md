# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Initial project setup
- FastAPI backend with core sentiment analysis
- Streamlit dashboard interface
- MongoDB integration
- Multilingual support (English, Hindi)
- JWT authentication
- Docker support

### Changed
- Cleaned up test files and debug scripts
- Moved MongoDB credentials to environment variables

### Fixed
- Security: Removed hard-coded credentials from codebase

## [1.0.0] - 2025-12-10

### Initial Release

**Features:**
- Sentiment analysis using IndicBERT
- Text summarization with transformer models
- Contextual analysis and key insights extraction
- User authentication and role-based access
- Interactive dashboard
- REST API for programmatic access
- Docker containerization
- Support for multiple Indian languages

**Documentation:**
- README with quick start guide
- API documentation
- Contributing guidelines
- Docker setup instructions
